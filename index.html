AYNI-BOT

Sistema Robótico Inteligente para Monitoreo de Calidad del Agua

¿Qué es AYNI-BOT?

AYNI-BOT es un robot autónomo diseñado para monitorear cuerpos de agua utilizando sensores inteligentes e integración IoT.

El sistema analiza parámetros físico-químicos en tiempo real y permite generar monitoreo ambiental inteligente.

Parámetros de Monitoreo

pH

Acidez o alcalinidad del agua.

TDS

Sólidos totales disueltos.

Conductividad

Capacidad iónica del agua.

Salinidad

Concentración de sales.

Nitratos

Detección de contaminación agrícola.

GPS

Ubicación geográfica precisa.

Beneficios

Monitoreo en tiempo real
Integración IoT
Protección ambiental
Datos geolocalizados
