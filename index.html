AYNI-BOT

Sistema Robótico Inteligente para el Monitoreo de Calidad del Agua 💧

Plataforma Robótica Autónoma

AYNI-BOT es un sistema robótico autónomo diseñado para monitorear la calidad del agua mediante sensores inteligentes y comunicación IoT.

El sistema permite realizar mediciones en tiempo real en lagunas, reservorios, piscinas y cuerpos hídricos naturales.

Integra tecnologías de geolocalización GPS/GNSS, transmisión inalámbrica y análisis ambiental automatizado.
