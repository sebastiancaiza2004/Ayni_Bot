Descripción Técnica

AYNI-BOT es un sistema robótico flotante diseñado para el monitoreo continuo de parámetros fisicoquímicos del agua en tiempo real. El sistema integra sensores especializados, procesamiento embebido y transmisión IoT.

Arquitectura del Sistema

Módulo de Sensado
  • Sensor de pH
  • Sensor de Cloro
  • Sensor TDS
  • Sensor de Turbidez
  • Sensor de Temperatura
Módulo de Procesamiento
  • Microcontrolador embebido
  • Conversión ADC
  • Filtrado de señal
  • Calibración y validación de datos
Módulo de Comunicación
  • Transmisión IoT en tiempo real
  • Envío estructurado de datos
  • Visualización remota
Sistema Energético
  • Panel solar
  • Batería de litio recargable
  • Regulador de carga
  • Optimización energética

Funcionamiento

  1. El robot flota de forma estable.
  2. Los sensores capturan datos periódicamente.
  3. El microcontrolador procesa la información.
  4. Los datos se transmiten en tiempo real.

Aplicaciones

  • Monitoreo ambiental
  • Control de piscinas
  • Gestión hídrica
  • Proyectos educativos STEAM

Beneficios de AYNI-BOT

  • Monitoreo continuo 24/7
  • Datos en tiempo real
  • Reducción de inspecciones manuales
  • Mayor precisión en control de calidad
  • Sistema energéticamente autónomo
  • Escalable para futuras mejoras
